 Phishing URL Detector:-

A web-based application that detects whether a URL is **Safe** or **Phishing** using a combination of **Machine Learning** and **rule-based analysis**.

 Features:-

-  Detects phishing URLs in real-time  
-  Machine Learning model for prediction  
-  FastAPI backend for API handling  
-  Interactive frontend UI  
-  Confidence score for predictions  
-  Hybrid approach (ML + rules)  
-  History tracking (recent checks)  
-  Visualization using charts  

 Tech Stack:-

- Frontend: HTML, CSS, JavaScript  
- Backend:  FastAPI (Python)  
- ML Model: Scikit-learn  
- Visualization: Chart.js  
- Version Control: Git & GitHub
